# Citações em Documentos — ABNT NBR 10520/2023

Guia detalhado de acordo com o Manual de Normalização UFV 2025.

---

## Definições

| Termo | Definição |
|-------|-----------|
| **Citação** | Menção de uma informação extraída de outra fonte |
| **Citação direta** | Transcrição literal de parte da obra do autor consultado |
| **Citação indireta** | Texto baseado na obra do autor, com as palavras do redator (paráfrase) |
| **Citação de citação** | Citação de um documento ao qual não se teve acesso, baseada em citação feita por outro autor (`apud`) |

---

## Regras gerais de apresentação

- O sistema de chamada (autor-data ou numérico) deve ser **uniforme** em todo o documento.
- Nas citações, indicar: autor, ano de publicação e página(s) — para citações diretas.
- Supressões: `[...]`
- Interpolações, acréscimos ou comentários: `[texto]`
- Ênfase ou destaque: indicar `(grifo nosso)` ou `(grifo do autor)` após a citação.
- Erros do texto original: reproduzir com `[sic]` após o erro.
- Tradução do redator: indicar `(tradução nossa)` após a citação.

---

## Citação direta

### Curta (até 3 linhas)
- Inserida no próprio texto, entre **aspas duplas**.
- Aspas simples são usadas para indicar aspas dentro de aspas.

Exemplo:
> Segundo Silva (2020, p. 45), "o processo de aprendizagem é contínuo e dinâmico".

ou

> "O processo de aprendizagem é contínuo e dinâmico" (SILVA, 2020, p. 45).

### Longa (mais de 3 linhas)
- **Recuo de 4 cm** da margem esquerda.
- Fonte **menor** que a do texto (geralmente tamanho 10).
- Espaçamento **simples** entre linhas.
- **Sem aspas**.
- Separada do texto por uma linha em branco antes e depois.

Exemplo:
```
[texto normal]

        Segundo Silva (2020, p. 45):

        Aqui vai o texto da citação longa, recuado 4 cm da margem esquerda,
        com fonte menor, espaçamento simples e sem aspas. O texto continua
        até o fim da transcrição literal do trecho citado.

[texto normal continua]
```

---

## Citação indireta (paráfrase)

- Não requer aspas nem recuo.
- Indicar autor e ano; página é opcional (recomendada quando possível).
- Exemplo: `De acordo com Silva (2020), o processo de aprendizagem...`
- Exemplo: `O processo de aprendizagem é contínuo (SILVA, 2020).`

---

## Citação de citação (apud)

- Usar **somente quando o original não está disponível**.
- Formato: `AUTOR ORIGINAL apud AUTOR DA OBRA CONSULTADA, ano`
- Na lista de referências: incluir **apenas** a obra consultada (não o original).

Exemplo no texto:
> Segundo Freire (1996 apud SILVA, 2020, p. 30), "a educação é um ato de amor".

## Sistema autor-data

A chamada é feita pelo **sobrenome do autor** + **ano** entre parênteses.

### Formas de chamada

**Autor integrado na frase:**
```
Silva (2020) afirma que...
```

**Autor entre parênteses:**
```
...(SILVA, 2020).
```

**Com página (citação direta):**
```
(SILVA, 2020, p. 45)
Silva (2020, p. 45)
```

**Dois autores:**
```
(SILVA; SOUZA, 2020)
Silva e Souza (2020)
```

**Três ou mais autores:**
```
(SILVA et al., 2020)
Silva et al. (2020)
```

**Vários documentos do mesmo autor no mesmo ano:**
```
(SILVA, 2020a)
(SILVA, 2020b)
```
> Usar a mesma designação (a, b...) nas referências.

**Vários autores diferentes no mesmo parêntese:**
```
(SILVA, 2018; SOUZA, 2020; SANTOS, 2021)
```
> Ordem cronológica ou alfabética, de forma consistente.

**Entidade coletiva (pessoa jurídica):**
```
(ASSOCIAÇÃO BRASILEIRA DE NORMAS TÉCNICAS, 2011)
ou com sigla conhecida: (ABNT, 2011)
```

**Sem autor (entrada pelo título):**
```
(PEQUENA biblioteca do vinho, 2012)
```
> Usar as primeiras palavras do título em maiúsculas até o ponto de elipse ou o suficiente para identificar a obra.

## Sistema numérico

- A citação recebe um número, que é colocado **sobrescrito** ou entre parênteses, na ordem de aparecimento no texto.
- As referências são numeradas na mesma ordem.
- Uma vez atribuído um número a uma fonte, usar o mesmo número sempre que ela for citada novamente.

Exemplo:
```
O processo de aprendizagem é contínuo.¹

--- lista de referências ---
1. SILVA, João. Título. Local: Editora, 2020.
```

---

## Notas de rodapé

### Notas de referência
- Indicam as fontes consultadas ou remetem a outras partes do trabalho.
- Podem substituir o sistema de chamada no texto (sistema numérico).

### Notas explicativas
- Contêm comentários, esclarecimentos ou traduções que não podem ser inseridos no texto.
- Devem ser sucintas.

**Formatação das notas de rodapé:**
- Separadas do texto por filete de 5 cm a partir da margem esquerda.
- Fonte menor que a do texto.
- Espaçamento simples.
- Segunda linha abaixo da primeira letra da primeira palavra (para destacar o número/expoente).

Exemplo:
```
_________________________
¹ Veja-se como exemplo desse tipo de abordagem o estudo de Netzer (1976).
² Encontramos esse tipo de perspectiva na 2ª parte do verbete referido na nota anterior.
```

## Casos especiais

### Obra sem data
```
(SILVA, [2020?])
```

### Comunicação pessoal, informação verbal
```
(SILVA, informação verbal, 2020)
```
> Em nota de rodapé: dados disponíveis sobre a comunicação.

### Obra em fase de elaboração (no prelo)
```
(SILVA, no prelo)
```

### Documentos jurídicos (lei, decreto, etc.)
- Sistema autor-data: entrada pela **jurisdição** ou **entidade** + ano.
```
(BRASIL, 2020)
(SÃO PAULO, 2019)
```
