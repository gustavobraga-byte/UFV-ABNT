# Elaboração de Referências — ABNT NBR 6023/2018

Guia detalhado de acordo com o Manual de Normalização UFV 2025.

---

## Regras gerais

- Referências: espaço simples, alinhadas à margem esquerda, separadas entre si por linha em branco.
- Em notas de rodapé: alinhadas à margem esquerda, segunda linha abaixo da primeira letra da primeira palavra (sem espaço entre elas).
- Recurso tipográfico (negrito, itálico ou sublinhado) **uniforme** para o título em todas as referências.
- Documentos online: incluir `Disponível em: [URL]. Acesso em: [data].`
- Obras sem autoria identificada: entrada pelo **título** (primeira palavra e artigo inicial em MAIÚSCULAS).

---

## Indicação de responsabilidade (autores)

### Pessoa física

**Regra geral:** ÚLTIMO SOBRENOME, Prenome e outros sobrenomes.

**Até 3 autores:** listar todos separados por ponto e vírgula.
```
PASSOS, L. M. M.; FONSECA, A.; CHAVES, M. Título. Local: Editora, ano.
```

**4 ou mais autores:** listar todos, ou indicar apenas o primeiro + `et al.`
```
GAW, A. et al. Título. Local: Editora, ano.
```

**Sobrenomes hispânicos (dois sobrenomes):** entrar pelo conjunto dos dois sobrenomes.
```
GARCÍA MÁRQUEZ, Gabriel. Título. Local: Editora, ano.
```

**Com grau de parentesco (Júnior, Filho, Neto, Sobrinho):**
```
ASSAF NETO, Alexandre. Título. Local: Editora, ano.
```

**Com prefixos (D', La, De, Von...):**
```
D'AMBROSIO, Ubiratan. Título. Local: Editora, ano.
LA TORRE, Massimo. Título. Local: Editora, ano.
```

**Organizador, editor, coordenador:**
```
FERREIRA, Léslie Piccolotto (org.). Título. Local: Editora, ano.
MARCONDES, E.; LIMA, I. N. de (coord.). Título. Local: Editora, ano.
```

**Autoria desconhecida:**
```
PEQUENA biblioteca do vinho. Local: Editora, ano.
```

### Pessoa jurídica

Entrada pelo nome mais conhecido, por extenso ou abreviado.
```
ASSOCIAÇÃO BRASILEIRA DE NORMAS TÉCNICAS. ABNT NBR 14724: informação e documentação: trabalhos acadêmicos: apresentação. Rio de Janeiro: ABNT, 2011.
```

Órgão governamental: precedido do órgão superior ou jurisdição.
```
BRASIL. Ministério da Ciência e Tecnologia. Título. Local: Ministério, ano.
SÃO PAULO (Estado). Secretaria do Meio Ambiente. Título. Local: Secretaria, ano.
```

Municípios homônimos: acrescentar sigla do estado entre parênteses.
```
VIÇOSA (MG). Lei nº 2558/2016. [descrição]. Local: Editora, ano.
```

## Modelos de referências

### Monografia (livro) no todo

```
SOBRENOME, Nome. Título em itálico: subtítulo. Edição. Local: Editora, ano. páginas p.
```
Exemplo:
```
PEDROSA, Israel. Da cor à cor inexistente. 6. ed. Rio de Janeiro: L. Cristiano, 1995. 219 p.
```

### Trabalho acadêmico (TCC, monografia, dissertação, tese)

```
SOBRENOME, Nome. Título em itálico. Ano. X f. Tipo (Grau em Área) – Instituição, Cidade, ano.
```
Exemplo:
```
ACCORSSI, Aline. Materializações do pensamento social sobre a pobreza. 2011. 184 f. Tese (Doutorado em Psicologia) – Faculdade de Psicologia, PUCRS, Porto Alegre, 2011.
```

### Artigo de periódico impresso

```
SOBRENOME, Nome. Título do artigo. Nome do Periódico em itálico, Local, v. X, n. X, p. XX-XX, mês ano.
```
Exemplo:
```
BENNETTON, M. J. Terapia ocupacional e reabilitação psicossocial: uma relação possível. Revista de Terapia Ocupacional da Universidade de São Paulo, São Paulo, v. 4, n. 3, p. 11-16, mar. 1993.
```

### Artigo de periódico online

```
SOBRENOME, Nome. Título. Nome do Periódico, Local, v. X, n. X, p. XX-XX, mês ano. Disponível em: URL. Acesso em: dia mês. ano.
```

### Parte de livro (capítulo)

```
SOBRENOME, Nome. Título do capítulo. In: SOBRENOME, Nome (org.). Título do livro em itálico. Edição. Local: Editora, ano. p. XX-XX.
```
Exemplo:
```
REGO, L. L. B. O desenvolvimento cognitivo e a prontidão para a alfabetização. In: CARRARO, T. N. (org.). Aprender pensando. 6. ed. Petrópolis: Vozes, 1991. p. 31-40.
```

### Documento de acesso exclusivo em meio eletrônico (site)

```
AUTOR/ORGANIZAÇÃO. Título. Local, ano. Disponível em: URL. Acesso em: dia mês. ano.
```

### Evento (anais)

```
NOME DO EVENTO EM MAIÚSCULAS, número., ano, Local. [Anais/Título]. Local: Editora, ano.
```
Exemplo:
```
SIMPÓSIO DE INSTRUMENTAÇÃO E IMAGENS MÉDICAS, 3., 2007, São Carlos. [Anais]. São Carlos, SP: Escola de Engenharia de São Carlos-USP, 2007. 1 CD-ROM.
```

### Documento jurídico (legislação)

```
JURISDIÇÃO (ou ENTIDADE). Tipo, número, data. Ementa. Local: Editora, ano. Disponível em: URL. Acesso em: data.
```
Exemplo:
```
VIÇOSA (MG). Lei nº 2558/2016. Dispõe sobre o direito ao aleitamento materno e dá outras providências. Viçosa, MG: Sistema de Leis Municipais, 2017. Disponível em: leismunicipais.com.br. Acesso em: 22 jun. 2017.
```

### Redes sociais

```
AUTOR. Mensagem/postagem. [Local], data. Nome da rede: perfil. Disponível em: URL. Acesso em: data.
```
Exemplo:
```
ASSOCIAÇÃO BRASILEIRA DE NORMAS TÉCNICAS. Sistemas de armazenagem. São Paulo, 19 set. 2017. Facebook: ABNT Normas Técnicas @ABNTOficial. Disponível em: [URL]. Acesso em: 21 set. 2017.
```

### E-book

```
SOBRENOME, Nome. Título. Local: Editora, ano. E-book (X p.). ISBN XXXX. Disponível em: URL. Acesso em: data.
```

### Podcast

```
TÍTULO do podcast. [Locução de]: Nome. [S. l.]: Produtora, data. Podcast. Disponível em: URL. Acesso em: data.
```

## Data de publicação

- **Sem data:** usar copyright (`c1993`), data provável entre colchetes (`[1953?]`), data aproximada (`[ca. 1960]`), década certa (`[199-]`).
- **Meses:** abreviados no idioma original (jan., fev., mar., abr., maio, jun., jul., ago., set., out., nov., dez.).
- **Dois períodos:** separados por barra (`maio/dez. 1996`).

---

## Local e editora

- **Local desconhecido:** `[S. l.]`
- **Editora desconhecida:** `[s. n.]`
- **Ambos desconhecidos:** `[S. l.: s. n.]`
- **Duas editoras em locais diferentes:** indicar ambas com seus locais, separadas por ponto e vírgula.
- **Duas editoras no mesmo local:** separadas por dois pontos.

---

## Ordenação das referências

- **Sistema alfabético:** por sobrenome do primeiro autor; obras do mesmo autor em ordem cronológica crescente; substituir o nome por seis hífens (`______`) quando se repetir.
- **Sistema numérico:** numeração sequencial na ordem de citação no texto.
